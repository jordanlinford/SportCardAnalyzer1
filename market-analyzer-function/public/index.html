Sports Card Market Analyzer API

This is the API endpoint for the Sports Card Market Analyzer.

How to use:

Send a POST request to /api/text-search with a JSON body:

{
  "query": "justin jefferson 2020 prizm",
  "grade": "PSA 10",  // optional 
  "isRaw": false      // optional
}

The API will return eBay sales data and market analysis for the specified card.
